# 🎓 Digital Student ID Card

## 📌 Project Overview
The Digital Student ID Card is a simple application that generates a virtual ID card for students. It helps replace physical ID cards with a digital version that can be accessed anytime.

## 🚀 Features
- Generate student ID card digitally
- Display student details (Name, ID, Department, Photo)
- Easy to use interface
- Portable and secure
- Reduces need for physical cards

## 🛠️ Technologies Used
- Python
- Tkinter (for GUI)
